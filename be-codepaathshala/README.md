CodingJudge: Empowering Ed-techs to Do Better & Beyond
CodingJudge is a business-to-business (B2B) platform designed to empower educational technology companies (ed-techs) by providing them with advanced technological capabilities and automations to enhance their operations and foster growth.

Features
Compile Code
Compile your code effortlessly using our compiler, which supports over 10+ programming languages.

Test Code
Evaluate the efficiency and accuracy of your code using our pre-built test cases.

Submit Code
Submit your code seamlessly, track the process, and verify its acceptance.

Review Reports
Monitor your progress effectively through weekly reports, leaderboards, and scores.

Assessment Module (V2)
The legacy assessments app has been removed and replaced by assessment_V2.
Base path: /api/assessment_V2/
Key endpoints (see assessment_V2/urls.py for full list):
- details/ : Get assessment questions
- assessment-info/ : Get assessment metadata for a user
- submit-problem/ : Submit/compile a coding problem
- open-assessment-submission/ : Public/open assessment submission
- admin/assessment-list/ & admin/assessment-detail/ : Admin management
- get-assessments/ : List assessments
- get-problems/, get-single-mcq/, get-multi-mcq/, get-subjective-questions/
- create-assessment/, create-assessment-item/, get-assessment-items/
- get-sets/, create-set/, set-priority-order/
- admin/export-graded-assessments/
- get-student-report/
